## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/crindalwalt/TaskHive.git
```

2. Navigate to project directory
```bash
cd task-hive
```

3. Get dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
